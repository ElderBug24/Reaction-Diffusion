# Reaction-diffusion algorithm
A simulation of two virtual chemicals reacting and diffusing.
See https://en.wikipedia.org/wiki/Reaction–diffusion_system for a more detailed explanation.

This repo has a release with two executables: one named gpu and the other named cpu.
The gpu one uses the graphical processing unit of the computer (if one is available) and speeds up the calculations by a factor of almost 3.
The cpu one only uses the cpu, is much slower but more compatible.
